排序算法的PHP实现--冒泡排序

介绍

冒泡排序或许可以算是所有排序算法中最重要的一种。它是最基础的排序算法，效率比较差，同时也是最容易理解的排序算法。基本每次面试都会被要求写。

它的基本原理是这样的：从数组的第一个元素开始，依次和之后相邻的一个相比较，如果不满足排序的条件，则交换这两个相邻的元素。这样一次遍历之后，就让最大的元素排到了最后。

代码实现

由于每次循环都排好最后的一个元素，所以内层循环要从后向前来减少。

function bubble($list) {
    $len = count($list);
    for ($i = 0; $i < $len; $i++) {
        for ($j = 0; $j < $len-$i-1; $j++) {
            if ($list[$j] > $list[$j+1]) {
                swap($list, $j, $j+1);
            }
        }
    }
    return $list;
}

内层循环也可以写成这样：

function bubble1(array $list) {
    $len = count($list);
    for ($i = 0; $i < $len; $i++) {
        for ($j = $len-1; $j > $i; $j--) {
            if ($list[$j] < $list[$j-1]) {
                swap($list, $j-1, $j);
            }
        }
    }
    return $list;
}

优化

如果一个序列基本有序，例如（1,2,3,4,5,6,7,9,8），其实只要遍历一次就可以排好，但上面的代码一直遍历和比较，浪费资源。可以添加一个标志，如果在这次循环中没有发生元素的交换，那么这个序列就已经是有序的了，不需要比较和交换了。

function bubble2(array $list) {
    $len = count($list);
    $flag = true;
    for ($i = 0; $i < $len && $flag; $i++) {
        $flag = false;
        for ($j = $len-1; $j > $i; $j--) {
            if ($list[$j] < $list[$j-1]) {
                swap($list, $j, $j-1);
                $flag = true;
            }
        }
    }
    return $list;
}

附：swap函数的实现

交换两个变量的值也是面试中的常见题，在这里有我的一些总结。

这里给出上面代码中的swap函数的实现（注意$list参数是数组的引用）：

function swap(&$list, $i, $j) {
    $temp = $list[$i];
    $list[$i] = $list[$j];
    $list[$j] = $temp;
}
