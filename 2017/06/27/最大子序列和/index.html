最大子序列和

问题描述

给定一个整数序列，a0, a1, a2, …… , an（每项值可以为负数），求其中最大的子序列和。如果所有整数都是负数，那么最大子序列和为0。

例如：对于序列-2， 11， -4， 13， -5， –2。 所求的最大子序列和为20（从11到13，即从 a1 到 a3 ）。

解法思路

第一步

首先想到的是穷举法，从 a0 开始遍历，提取以 a0 为首的子序列，计算它的和，并记录最大值，如果之后有比当前最大值的出现，则更新最大值。

function maxSubSum(array $list) {
    $len = count($list);
    $max = 0;
    for ($i = 0; $i < $len; $i++) {
        $sum = 0;
        for ($j = $i; $j < $len; $j++) {
            $sum += $list[$j];
            if ($sum > $max) $max = $sum;
        }    
    }
    return $max;
}

分析：

  1. 两层循环嵌套，时间复杂度为O(n^2)；
  2. 包含重复计算。例如：当i=1时，要计算a1+a2+a3+a4+……；当i=2时，要计算a2+a3+a4+……；其中a2+a3+a4+……是重复的。

第二步

通过观察，我们可以发现：如果这个序列中有正数，那么最大的子序列肯定不会是以一个负数开头。所以我们可以对在思路一的基础上进行改进。

function maxSubSum(array $list) {
    $max = 0;
    $len = count($list);
    for ($i = 0; $i < $len; $i++) {   //从第i个开始
        if ($list[$i] > 0) {
            $sum = 0;
            for ($j = $i; $j < $len; $j++) {
                $sum += $list[$j];
                if ($max < $sum) {
                    $max = $sum;
                }
            }
        }
    }
    return $max;
}

第三步

此时，我们的子序列肯定是以一个正数开头的。再次观察可以得出：任何和为负的子序列，不可能作为最优子序列的前缀。

也就是说，序列以正数开头 a[i]>0，所以若 a[i] 到 a[j] 之间元素的序列和 sum <= 0 时，则 i+1 和 j 之间元素不会为最优子序列的前缀，可以让 i = j，即不需要判断在 i 和 j 之间元素开头，直接重置 sum, 计算 a[j] 开头的子序列就可以。

function maxSubSum(array $list) {
    $max = 0;
    $len = count($list);
    for ($i = 0; $i < $len; $i++) {   //从第i个开始
        if ($list[$i] > 0) {
            $sum = 0;
            for ($j = $i; $j < $len; $j++) {
                $sum += $list[$j];
                if ($sum < 0) {
                    $i = $j;
                    $sum = 0;    //重置sum
                }elseif ($max < $sum) {
                    $max = $sum;
                }
            }
        }
    }
    return $max;
}

第四步

再次观察，可以得到：如果一个子序列和 sum > 0（其中 sum 是从第 i 项到第 j 项的和），那么序列一定不会以 a[i] 和 a[j] 之间的项开始。

因为 sum 第一个元素 a[i] 是大于0的，并且上一步保证了如果子序列和小于0，则退出，所以既然可以加到 a[j],说明 a[i] 开始到 a[j] 的所有子序列和都是大于0的。

也就是说，序列一定不会以 a[i] 和 a[j] 之间的项 a[m] 开始，因为从 a[i] 到 a[m-1] 的和是正数，从 a[m] 开始肯定没有从 a[i] 开始大。

function maxSubSum(array $list) {
    $max = 0;
    $len = count($list);  
    for ($i = 0; $i < $len; $i++) {   //从第i个开始
        if ($list[$i] > 0) {
            $sum = 0;
            for ($j = $i; $j < $len; $j++) {   
                $sum += $list[$j];
                if ($sum > 0) {
                    $i = $j;
                    if ($sum > $max) {
                        $max = $sum;
                    }
                }else {   //sum<0
                    $i = $j;
                    $sum = 0;
                }
            }
        }
    }
    return $max;
}

第五步

优化内层循环:

判断分支可以改变一下，去除嵌套分支结构；

判断的每个分支中有共同部分 i=j，可以抽取出来，放在公共部分；

function maxSubSum(array $list) {
    $max = 0;
    $len = count($list);
    for ($i = 0; $i < $len; $i++) {   //从第i个开始
        if ($list[$i] > 0) {
            $sum = 0;
            for ($j = $i; $j < $len; $j++) { 
                $sum += $list[$j];
                if ($sum > $max) {
                    $max = $sum;
                }elseif ($sum < 0) {
                    $sum = 0;
                }
                $i = $j;
            }
        }
    }
    return $max;
}

我们可以发现，内层循环的循环变量 j 和外层循环的循环变量 i 同步增长，那么，外层循环是不是可能没有存在的必要？

在这里到底能不能去除外层循环，取决于外层循环中是否有额外的工作要做。

这里的额外工作是 if(a[i] > 0) 判断语句，其实内层循环的 if(sum < 0) 的判断已经替代了 if(a[i] <= 0 ) 的工作。因为 sum 第一步是由 a[j] 得到的，也就是 a[i]，所以对它的判断也是对 a[i]的判断。

function maxSubSum(array $list) {
    $max = 0;
    $len = count($list); 
    $sum = 0;
    for ($j = 0; $j < $len; $j++) {  
        $sum += $list[$j];
        if ($sum > $max) {
            $max = $sum;
        }elseif ($sum < 0) {
            $sum = 0;
        }
    }
    return $max;
}

终于，我们达到了最优解：只有一层循环，时间复杂度为O(n)。常量空间，线性时间。

数组值全为负数时

上面我们的要求是，如果全为负数，则返回 0，现在不加这个条件，也就是如果全为负数，也是找出它的最大子序列值（其实就是最大的那个负数），应该怎么办呢？

我们只需要修改 $max 的初值即可，因为之前设定它为 0，数组全为负时，不会更新它。我们设置它为第一个数组值即可。

function maxSubSum(array $list) {
    $len = count($list); 
    $sum = 0;
    $max = $list[0];
    for ($j = 0; $j < $len; $j++) {  
        $sum += $list[$j];
        if ($sum > $max) {
            $max = $sum;
        }elseif ($sum < 0) {
            $sum = 0;
        }
    }
    return $max;
}
