排序算法的PHP实现--插入排序

介绍

插入排序的原理是这样的：假设从小到大排序，从第二个元素开始，如果它比第一个小，那么就把它插入到第一个元素前（其实是把第一个元素向后移动，把第二个元素放在空出的位置）。

同样的，在排第n个元素时，前n-1个元素组成的序列已经是有序的。我们要做的就是找到第n个元素应该插入的位置，然后向后移动这个位置之后的元素，接着把这个元素插入这个位置。

插入排序的时间复杂度最好是o(n)，最差是o(n2)。

代码实现

function insertion_sort(array $list) {
    $len = count($list);
    for ($i = 1; $i < $len; $i++) {
        if ($list[$i] < $list[$i-1]) {
            $temp = $list[$i];
            for ($j = $i-1; $j >= 0 && $list[$j] > $temp; $j--) {
                $list[$j+1] = $list[$j];
            }
            $list[$j+1] = $temp;
        }
    }
return $list;
}
