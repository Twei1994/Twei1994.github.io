Apache Rewrite常见实例

前言

在Apache实现伪静态–Rewrite模块学习这里，介绍了用 Apache 实现 URL 重写的相关知识，这里给出几个具体的实例，加深理解。

正文

伪静态

应用场景：实现伪静态的时候

规则：输入 http://www.twei.site/article/12 访问 http://www.twei.site/article?id=12

配置：

RewriteEngine On
RewriteRule ^article/([0-9]+)$ article?id=$1

注：也可以用 \d 表示 [0-9]

增加www

应用场景：

规则：输入 test.com 访问 www.test.com

配置： 

RewriteEngine On
RewriteCond %{HTTP_POST} ^test.com [NC]
RewriteRule ^(.*)$ http://www.test.com/$1 [R=301]

多域名指向主域名

应用场景：如果网站有多个域名，而又不希望分散权重，则可以通过开启Rewrite设置，将多个域名都 301 跳转到主域名之下

规则：其中 www.metooshow.com 为主域名，而 mituxiu.com、www.mituxiu.com 和 metooshow.com 都为希望被 301 跳转的域名，也就是访问后面 3 个都跳转至第一个

配置：

RewriteEngine On
RewriteCond %{HTTP_HOST} ^mituxiu.com [NC,OR]
RewriteCond %{HTTP_HOST} ^www.mituxiu.com [NC,OR]
RewriteCond %{HTTP_HOST} ^metooshow.com [NC]
RewriteRule ^(.*)$ http://www.metooshow.com/$1 [L,R=301]

二级域名

应用场景：我们有多个二级域名，每个对应网站的一个方面，如 article.twei.site 对应着网站的文章， bbs.twei.site 对应着网站的论坛等等，但在服务器上它们是通过一个 module 的参数区分的。这时我们就可以用 URL 重写实现二级域名的对应。

规则：输入 xxx.twei.site 访问 twei.stie/index.php?module=xxx

配置：

# 打开URL Rewrite功能 
RewriteEngine On
# 匹配条件，如果用户输入的URL中主机名是类似 xxxx.twei.site 就执行 Rule     
RewriteCond %{HTTP_HOST} ^[\w\d]+\.twei.site$
# 把用户输入完整的地址（GET方式的参数除外）作为参数传给下一个规则，[C]是Chain串联下一个规则的意思 
RewriteRuel ^(.+) %{HTTP_HOST}$1 [C]
# 解析用户输入的URL地址，把主机名中的 xxx 作为名为 module 的参数传给 http://twei.site/index.php?，并在后面跟上用户输入的 GET 方式的传入参数。并指明这是最后一条规则（[L]规则）
RewriteRule ^([\w\d]+)\.twei.site(.*)$ http://twei.site/index.php?moudle=$1&%{QUERY_STRING} [L]

防盗链

应用场景：盗链包括图片盗链、音乐或视频文件(如 mp3、Flash 等)盗链。所谓盗链，是指对方网站直接链接本网站上的文件，而不是将其置于自己的服务器上。一般而言，盗链的对象大多为较耗带宽的大体积文件，如图片、视频等。不仅服务器带宽被无任何回报地占用，而且往往会在很大程度上影响网站的访问速度。

规则：除了指定的网站，其他网站访问我们的图片都返回一个 error.gif

配置：

RewriteEngine On
# 允许空的来源，即允许用户在浏览器中直接输入图片地址时显示图片
RewriteCond %{HTTP_REFERER} !^$ [NC]
# 允许的几个网站，自身，百度，谷歌
RewriteCond %{HTTP_REFERER} !twei.site [NC]
RewriteCond %{HTTP_REFERER} !google.com [NC]
RewriteCond %{HTTP_REFERER} !baidu.com.com [NC]
# 如果上面都满足，形成盗链，返回错误图片
RewriteRule .*\.(gif|jpg|png)$ /allow/error.gif [R,NC,L]

注：类型不只图片，还可以是视频、压缩包、Flash等；处理的结果也不限于返回错误图片，可以跳转到首页等
