Web页面设计

POST和GET传输的最大容量分别是多少？

常见的一种说法是：GET方法传递数据，必须控制在1MB之内；POST方式则没有限制。

然而，我们都知道，GET是通过URL提交数据。实际上，URL不存在参数上限的问题，HTTP协议规范没有对URL长度进行限制。这个限制是特定的浏览器及服务器对它的限制。传统IE对URL长度的限制是2083字节(2K+35)。对于其他浏览器，如FireFox、Chrome等，理论上没有长度限制，其限制取决于操作系统的支持。

经过实际测试

在IE8 下的URL地址总长度为：4076，超过该长度会自动忽略后面的内容；

在firefox 25下的URL地址总长度可以达到:7530，超过该长度会访问错误；

在chrome 29.0.1547.62 的最大总长度达到:7675，超过该长度会访问错误；

这篇博客分析得更清楚，所以正确的答案应该是这样的：

不管是GET方式和是POST方式，都没有传输方式的限制。所谓的请求长度限制是由浏览器和 WEB 服务器的设置决定的，各种浏览器和 WEB 服务器的设定均不一样，这取决于各个浏览器厂家的规定或者根据 WEB 服务器的处理能力来设定。

如何通过form表单控制上传文件的大小？

<form enctype="multipart/form-data" action="upload.php" method="post">
<input type="hidden" name="MAX_FILE_SIZE" value="100000">
<input name="file" type="file">　　
<input type="submit" value="上传文件">
</form>

通过隐藏域并设置MAX_FILE_SIZE的值来控制上传文件的大小。

如何设置form表单中的只读属性？

可以通过readonly和disabled两种方式。

<form action="get">
    <input type="text" readonly="true" value="aaa" name="name">
    <input type="text" disabled="true" value="aaa" name="pass">
    <input type="submit">
</form>

这页面中显示是这样的：





都可以实现限制编辑，只读的功能，但二者也是有区别的：

  • 如果一个输入项的 disabled="true"，则该表单输入项不能获取焦点（整个 input 会变成灰色），用户的所有操作（鼠标点击和键盘输入等）对该输入项都无效，最重要的一点是当提交表单时，这个表单输入项将不会被提交；
  • 而 readonly 只是针对文本输入框（input(text / password) 和 textarea）的输入项，如果设为 true，用户只是不能编辑对应的文本，但是仍然可以聚焦焦点，并且在提交表单的时候，该输入项会作为form的一项提交。

在什么情况下，$name和$_POST[‘name’]可以通用？

在历史久远的 PHP4.2.0 版本之前，php.ini中有 register_globals 这样一个配置项。在它设置为 on 的时候，可以之间通过表单项的名称来获取表单元素的值。

<form method="post">
    <input type="text" name="user_name">
    <input type="password" name="user_pass">
    <input type="submit" value="login">
</form>

在开启了 register_globals 时，就可以之间使用 $user_name 和 $user_pass来接收表单的值，而不需要使用 $_POST['user_name'] 和 $_POST['user_pass']。

这种方式会带来一定的安全隐患。如果在变量没有初始化的情况下，通过在URL后增加一个参数就可以创建一个PHP变量，这可能会使业务绕过某些判断，从而达到窃取信息的目的。

这个配置自 PHP 5.3.0 起废弃并将自 PHP 5.4.0 起移除。

如何解决IE6的双倍边距问题？

这是CSS中的一个经典问题，所谓的IE6双倍边距就是指当元素有 float 属性，又有 margin 属性时，在IE6下面显示的 margin 的值是设置值的两倍。

解决方法就是添加属性：display: inline。

如何解决超链接点击之后，hover样式失效的问题？

必须按照固定顺序设定超链接的属性:link --> visited --> hover --> active

a:link {color: #FF0000} /* 未访问的链接 */
a:visited {color: #00FF00} /* 已访问的链接 */
a:hover {color: #FF00FF} /* 当有鼠标悬停在链接上 */
a:active {color: #0000FF} /* 被选择的链接 */

a:hover 必须位于 a:link 和 a:visited 之后，a:active 必须位于 a:hover 之后！

如何定义一个高度为1px的DIV?

利用行高和字体大小即可。

div {line-height: 1px; font-size: 0}

Javascript中获得输入焦点的函数？

focus()函数，可以用于用户填写错误时，自动将光标指向第一个出错的文本框中。

<head>
    <meta charset="utf-8">
    <title>self_focus</title>
    <script type="text/javascript">
        window.setTimeout(function(){
            document.getElementById("text_2").focus();
        }, 1000);
    </script>
</head>
<body>
    <input type="text" id="text_1" />
    <input type="text" id="text_2" />
</body>

以上代码会在窗口打开1s后，将光标自动置于第二个文本框中。

使用Jquery替换元素

两个函数，replaceWith() 和 replaceAll()。

$(document).ready(function(){
    $('div').click(function(){
        var title = $("<span>bbb</span>");
        $('div').replaceWith(title);
    });
});

点击 DIV 时将 DIV 替换成 span。

用Jquery获取某一个元素的位置

两个函数：offset() 和 position()，区别在于 offset() 获取到的位置是在当前视窗的相对偏移，而 position()是相对于父元素的偏移。另外，通过 offset() 函数可以重新设置元素的位置。

var top = $('div').offset().top;
var left = $('div').offset().left;

实现一个图片轮播效果

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style>
    ul {list-style: none; position: absolute;}
    li {position: absolute;}
    </style>
    <script src="http://libs.baidu.com/jquery/2.1.1/jquery.min.js"></script>
    <script type="text/javascript">
    $(document).ready(function(){
        $(".change ul li:not(:first)").hide();
        setInterval(function(){
            if ($(".change ul li:last").is(":visible")) {
                $(".change ul li:first").fadeIn("slow");
                $(".change ul li:last").hide();
            }else{
                $(".change ul li:visible").next().fadeIn("slow");                
            };
        },2500);
    });
    </script>
</head>
<body>
    <div class="change">
        <ul>
            <li><img src="1.jpg" width="350px" height="200px"></li>
            <li><img src="2.jpg" width="350px" height="200px"></li>
            <li><img src="3.jpg" width="350px" height="200px"></li>
            <li><img src="4.jpg" width="350px" height="200px"></li>
        </ul>
    </div>
</body>
</html>

几个关键点需要注意：

  • li的position必须设置为absolute，目的是让图片都重叠起来；
  • jquery中选择器的使用，:not()表示除某个元素之外的其他元素，:visible表示当前为可见的元素；
  • is(":visible") 和 next() 的使用；
