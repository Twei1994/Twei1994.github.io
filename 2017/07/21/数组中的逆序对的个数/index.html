数组中的逆序对的个数

题目描述

在数组中的两个数字如果前面一个数字大于后面的数字，则这两个数字组成一个逆序对。输入一个数组，求出这个数组中的逆序对的总数。

例如，有一个数组为 Array[0..n] 其中有元素 a[i]，a[j]。如果 当 i < j 时，a[i] > a[j], 那么我们就称（a[i],a[j]）为一个逆序对。

在数组 {7,5,6,4} 中一共存在 5 对逆序对，分别是 (7,6),(7,5),(7,4),(6,4),(5,4)。

思路解析

在第一次遇到这个问题的时候，我首先想到了“求数组中最大子序列和”那道题，可以通过固定一个元素改变另一个来判断。不过这样的时间复杂度为 o(n2)。

然后想到，逆序对的个数就是对这个数组进行排序时元素交换的次数。前提是，这个排序过程是稳定的，因为不稳定的排序算法（如选择排序）不能保证每对逆序的元素都交换一次。

那么，常见的稳定排序有插入排序，冒泡排序和归并排序。在这几种排序算法的基础上，都可以实现求逆序对个数。其中，冒泡排序和插入排序的复杂度为 o(n2)，归并排序的复杂度为 o(nlogn) 。

代码实现

利用冒泡排序

交换的次数就是逆序对的个数。

function inversePairs_bubble(array $list) {
    $len = count($list);
    $count = 0;
    for ($i = 0; $i < $len; $i++) { 
        for ($j = $len-1; $j > $i; $j--) { 
            if ($list[$j] < $list[$j-1]) {
                list($list[$j], $list[$j-1]) = array($list[$j-1], $list[$j]);
                $count++;
            }
        }
    }
    return $count;
}

利用插入排序

逆序对的个数就是元素向后移动的次数。

function inversePairs_insertion(array $list) {
    $len = count($list);
    $count = 0;
    for ($i = 1; $i < $len; $i++) { 
        if ($list[$i] < $list[$i-1]) {
            $temp = $list[$i];
            for ($j = $i-1; $j >= 0 && $list[$j] > $temp; $j--) { 
                $list[$j+1] = $list[$j];
                $count++;
            }
            $list[$j+1] = $temp;
        }
    }
    return $count;
}

利用归并排序

function inversePairs_merge(array $list) {
    $len = count($list);
    $res = array();
    $count = Msort($list, $res, 0, $len-1);
    unset($res);
    return $count;
}

重点就是注释的地方，分成左右两部分后，开始归并。从头开始比较两个序列，如果右边小于左边说明是逆序对，那么左边的从 i 到最后的 m 元素都比后面的大，所以逆序对的个数为 m-i+1 个。

function Msort(&$list, &$res, $i, $j) {
    if ($i == $j) {
        $res[$i] = $list[$j];
    }else{
        $m = floor(($i+$j)/2);
        $temp = array();
        $left = Msort($list, $temp, $i, $m);
        $right = Msort($list, $temp, $m+1, $j);

        $count = 0;
        for ($k = $i, $l = $m+1; $i <= $m && $l <= $j; $k++) { 
            if ($temp[$i] <= $temp[$l]) {
                $res[$k] = $temp[$i++];
            }else{
                $res[$k] = $temp[$l++];
                $count += $m-$i+1;   // 重点，从开始比较左右序列，如果右边小说明是逆序对，那么左边从 i 到最后的 m 都比后面的大
            }
        }
        if ($i <= $m) {
            for ($s = 0; $s <= $m-$i; $s++) { 
                $res[$k+$s] = $temp[$i+$s];
            }
        }
        if ($l <= $j) {
            for ($s = 0; $s <= $j-$l; $s++) { 
                $res[$k+$s] = $temp[$l+$s];
            }
        }
        return $count+$left+$right;
    }
}
