层序输出二叉树

题目描述

从上往下打印出二叉树的每个结点，同一层的结点按照从左到右的顺序打印。

例如输入

      8
    /  \
   6    10
  /\     /\
 5  7   9  11

输出 8 6 10 5 7 9 11

思路解析

基本的解法是通过一个队列（queue）。从根节点开始，如果队列非空，那么就输出一个节点并将它的左右子节点（如果存在的话）加入队列。这样当队列为空时，就完成了层序输出二叉树。

代码实现

首先，给出二叉树节点的定义：

class Node {
    public $value;
    public $left;
    public $right;

    public function __construct($value) {
        $this->value = $value;
        $this->left = null;
        $this->right = null;
    }
}

然后是层序输出二叉树的算法（利用 array_push() array_shift() 这两个函数将数组作为队列使用）：

function scanBTbyLevel(Node $root = null) {
    if (!is_null($root)) {
        $queue = array();
        array_push($queue, $root);
        while (!empty($queue)) {
            $cur = array_shift($queue);
            echo $cur->value." ";
            if ($cur->left) {
                array_push($queue, $cur->left);
            }
            if ($cur->right) {
                array_push($queue, $cur->right);
            }
        }
    }
}

题目描述

层序遍历二叉树的一个升级版，要求输出时按层换行，也就是输入

      8
    /  \
   6    10
  /\     /\
 5  7   9  11

输出为

8
6 10
5 7 9 11

思路解析

我们可以基于上一题的队列接着想。要求按层换行，就需要我们记录每一层的节点个数。当这层输出的个数等于节点数时，说明这层输出完了，输出一个换行符。所以我们还需要一个变量来记录这层剩余节点个数。

总之，需要在原来的基础上新增两个变量，一个记录这层剩余节点个数，另一个记录下一层的节点数。

代码实现

function scanBTByLevel(Node $root = null) {
    if (!is_null($root)) {
        $left = 1;     // 当前层剩余节点个数，初始为1，根节点
        $next = 0;     // 下一层节点个数，初始为0
        $queue = array();
        array_push($queue, $root);
        while (!empty($queue)) {
            $cur = array_shift($queue);
            echo $cur->value." ";
            $left--;
            if ($cur->left) {
                array_push($queue, $cur->left);
                $next++;
            }
            if ($cur->right) {
                array_push($queue, $cur->right);
                $next++;
            }
            if ($left == 0) {    // 当前层剩余节点为0，更新 left 和 next值
                echo "<br>";
                $left = $next;
                $next = 0;
            }
        }
    }
}

题目描述

上一道题的一个变种，要求输出不仅按层换行，而且每层是顺序不一样（之字形），也就是输入

      8
    /  \
   6    10
  /\     /\
 5  7   9  11

输出为

8
10 6
5 7 9 11

思路解析

这时就不能采用队列了。如果根节点算第 0 层的话，那么第一层的 6 和 10 需要先放入 10 再放入 6，队列中的内容为 10，6（根节点 8 已经不在队列中了）。此时要求我们放入 5 和 7，但是队列的头部元素是 10，所以不可能到达 6 的子节点。

那么，我们只能使用两个容器来存数据了，奇数和偶数层各用一个。注意到输入顺序和输出的顺序是相反的，可以考虑用栈。而且奇数层和偶数层的输入顺序也不同，所以还需要判断当前层是奇数还是偶数。

以上面的二叉树为例，首先根节点入栈 0，然后出栈，这时判断它的左右节点是否存在，按照先左后右的顺序加入栈 1。栈 1 的内容为 6，10。栈 1 执行出栈操作，值为 10 的节点出栈，这时要按照先右后左的顺序将它的子节点放入栈 0 中。

总之，需要两个栈分别存奇数和偶数层节点，偶数层时入栈顺序先左后右，奇数层入栈顺序先右后左，两个栈都为空时结束。

代码实现

function scanBTByZ(Node $root = null) {
    if (!is_null($root)) {
        $current = 0;
        $next = 1;
        $stack = array(
            0 => array(),
            1 => array()
            );
        array_push($stack[0], $root);
        while (!empty($stack[0]) || !empty($stack[1])) {
            $cur = array_pop($stack[$current]);   // 从当前栈中取出一个元素
            echo $cur->value." ";
            if ($current == 0) {     // 当前层为偶数层，先左后右
                if ($cur->left) {
                    array_push($stack[$next], $cur->left);
                }
                if ($cur->right) {
                    array_push($stack[$next], $cur->right);
                }
            }else{
                if ($cur->right) {
                    array_push($stack[$next], $cur->right);
                }
                if ($cur->left) {
                    array_push($stack[$next], $cur->left);
                }
            }
            if (empty($stack[$current])) {     // 当前层节点为空，更新current和next
                echo "<br>";
                $current = 1-$current;
                $next = 1-$next;
            }
        }
    }
}

定义了两个栈 stack[0] 和 stack[1]，打印当前层的节点时，将下一层的节点保存在另一个栈中。用 current 和 next 表示当前层和下一层为奇数还是偶数。当前栈为空时，输出换行符并交换 current 和 next 值，输出下一层。
