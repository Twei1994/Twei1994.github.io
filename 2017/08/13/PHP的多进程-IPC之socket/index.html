正文

PHP 实现进程通信（IPC：InterProcess Communication）的方式有以下几种：共享内存、信号量、消息队列、信号、管道和 socket。

这篇的内容是 PHP 通过 socket 实现进程通信。使用消息队列需要 PHP 安装 sockets 扩展（在 congfigure 时使用 --enable-sockets 配置，在 php.ini 中没有相关的扩展）。

Socket

之前我们所说的进程间通信，都属于同一计算机上的进程。如果两个进程不在同一台计算机，那么它们之间的通信必须使用 socket（当然，Socket也可以实现本机 IPC）。

要理解 socket，我们首先要从 TCP/IP 协议族讲起。TCP/IP（Transmission Control Protocol/Internet Protocol）即传输控制协议/网间协议，是一个工业标准的协议集，它是为广域网（WANs）设计的。

TCP/IP 协议族是一系列网络协议的总称，包括链路层、网络层和运输层。用户进程属于应用层（如浏览器使用的 HTTP 协议，还有 FTP 等）。不同主机的应用层的数据依据运输层的 TCP/UDP 协议，网络层的 IP 协议和数据链路层的 ARP 协议等，实现数据的交换。

那么 socket 有什么用呢？在整个通信过程中扮演什么角色呢？

上图显示了 socket 的位置。Socket 是应用层与 TCP/IP 协议族通信的中间软件抽象层，相当于是一组接口。它把复杂的 TCP/IP 协议族隐藏在后面。用户编程面向的就是 Socket，而不是复杂的 TCP/IP 协议族中的各种协议。不需要我们自己实现，Socket 去组织数据，以符合指定的协议。

它的执行过程类似于 TCP 协议的通信过程，也是分为连接的建立和断开（Socket 可以实现多种协议，不只是 TCP 一种）。

服务器端先初始化 Socket，然后与端口绑定（bind），对端口进行监听（listen），调用 accept 阻塞，等待客户端连接。在这时如果有个客户端初始化一个 Socket，然后连接服务器（connect）。如果连接成功，这时客户端与服务器端的连接就建立了。客户端发送数据请求，服务器端接收请求并处理请求，然后把回应数据发送给客户端，客户端读取数据，最后关闭连接，一次交互结束。

相关函数

// 创建一个实现某协议的socket
$socket = socket_create($domain, $type, $protocol)

// 绑定某IP+prot到该socket
socket_bind($socket, $address, $port)

// 监听某个socket，在server端使用
socket_listen($socket)

// 确认socket连接请求，返回一个socket连接（$msgsock）或FALSE（失败）
$msgsock = socket_accept($socket)

// 向socket连接中写入数据$msg
socket_write($msgsock, $msg, strlen($msg))

// 从socket连接中读取数据
socket_read($msgsock, 2048, PHP_NORMAL_READ)

// 关闭socket资源或者socket连接
socket_close($socket/$msgsock)

案例1

实现一个简单的 TCP/IP 服务器，功能是接受客户端的数据并返回。

<?php
error_reporting(E_ALL);

/* Allow the script to hang around waiting for connections. */
set_time_limit(0);

/* Turn on implicit output flushing so we see what we're getting
 * as it comes in. */
ob_implicit_flush();

$address = '192.168.1.116';
$port = 10000;

if (($sock = socket_create(AF_INET, SOCK_STREAM, SOL_TCP)) === false) {
    echo "socket_create() failed: reason: " . socket_strerror(socket_last_error()) . "\n";
}

if (socket_bind($sock, $address, $port) === false) {
    echo "socket_bind() failed: reason: " . socket_strerror(socket_last_error($sock)) . "\n";
}

if (socket_listen($sock, 5) === false) {
    echo "socket_listen() failed: reason: " . socket_strerror(socket_last_error($sock)) . "\n";
}

do {
    if (($msgsock = socket_accept($sock)) === false) {
        echo "socket_accept() failed: reason: " . socket_strerror(socket_last_error($sock)) . "\n";
        break;
    }
    /* Send instructions. */
    $msg = "\nWelcome to the PHP Test Server. \n" .
        "To quit, type 'quit'. To shut down the server type 'shutdown'.\n";
    socket_write($msgsock, $msg, strlen($msg));

    do {
        if (false === ($buf = socket_read($msgsock, 2048, PHP_NORMAL_READ))) {
            echo "socket_read() failed: reason: " . socket_strerror(socket_last_error($msgsock)) . "\n";
            break 2;
        }
        if (!$buf = trim($buf)) {
            continue;
        }
        if ($buf == 'quit') {
            break;
        }
        if ($buf == 'shutdown') {
            socket_close($msgsock);
            break 2;
        }
        $talkback = "PHP: You said '$buf'.\n";
        socket_write($msgsock, $talkback, strlen($talkback));
        echo "$buf\n";
    } while (true);
    socket_close($msgsock);
} while (true);

socket_close($sock);
?>

保存为 server.php，在命令行执行 php -f server.php，进入等待状态。此时，在另外的 terminal 中使用命令 netstat -ano | grep 10000 可以发现此时系统中有一个占用了 10000 端口号的 TCP 服务。

使用 telnet 192.168.1.116 10000 可以模拟客户端连接到我们刚才建立的 socket 服务端，发送信息 hello world，可以获取到服务端的输出，通过 quit 指令也可以断开连接。

案例2

实现一个简单的，一次性的 HTTP 客户端。 它只是连接到一个页面，提交一个 HEAD 请求，输出回复，然后退出。

<?php
error_reporting(E_ALL);

echo "<h2>TCP/IP Connection</h2>\n";

/* Get the port for the WWW service. */
$service_port = 10000;

/* Get the IP address for the target host. */
$address = "192.168.1.116";

/* Create a TCP/IP socket. */
$socket = socket_create(AF_INET, SOCK_STREAM, SOL_TCP);
if ($socket === false) {
    echo "socket_create() failed: reason: " . socket_strerror(socket_last_error()) . "\n";
} else {
    echo "OK.\n";
}

echo "Attempting to connect to '$address' on port '$service_port'...";
$result = socket_connect($socket, $address, $service_port);
if ($result === false) {
    echo "socket_connect() failed.\nReason: ($result) " . socket_strerror(socket_last_error($socket)) . "\n";
} else {
    echo "OK.\n";
}

$in = "HEAD / HTTP/1.1\r\n";
$in .= "Host: www.example.com\r\n";
$in .= "Connection: Close\r\n\r\n";
$out = '';

echo "Sending HTTP HEAD request...";
socket_write($socket, $in, strlen($in));
echo "OK.\n";

echo "Reading response:\n\n";
while ($out = socket_read($socket, 2048)) {
    echo $out;
}

echo "Closing socket...";
socket_close($socket);
echo "OK.\n\n";
?>
