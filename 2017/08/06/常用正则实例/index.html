常用正则实例

前言

这里列举了一些常用的正则匹配实例。

正文

IP地址

0.0.0.0 到 255.255.255.255

$pattern = "/^(((\d)|([1-9]\d)|(1\d{2})|(2[0-4]\d)|(25[0-5]))\.){3}((\d)|([1-9]\d)|(1\d{2})|(2[0-4]\d)|(25[0-5]))$/";

也就是把 0~255 分成 0~9、10~99、100~199、200~249、250~255来匹配。

不能用 1?\d{1,2} 是因为这样会认为 00 也是合法的。

邮箱

$pattern = "/^\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$/"

手机号

$pattern = "/^(13[0-9]|14[5|7]|15[0|1|2|3|5|6|7|8|9]|18[0|1|2|3|5|6|7|8|9])\d{8}$/"

以前面的 3 位数开头，再加上 8 位数字

URL

$ip_pattern = "(((\d)|([1-9]\d)|(1\d{2})|(2[0-4]\d)|(25[0-5]))\.){3}((\d)|([1-9]\d)|(1\d{2})|(2[0-4]\d)|(25[0-5]))"; 

$pattern = "/^(http|https|ftp):\/\/((".$ip_pattern.")|([a-zA-Z\-]+[\.a-zA-Z\-]+))(:\d{1,4})?[\w\/\.\?=&;%@#\+\-]*$/";

关于该正则表达式的说明：

  • 该正则表达式匹配的字符串必须以 http://、https://、ftp://开头；

  • 该正则表达式能匹配域名或者 IP 地址；（如：http://www.baidu.com 或者 http://192.168.1.1）

  • 该正则表达式能匹配到 URL 的末尾，即能匹配到子 URL；（如能匹配：http://www.baidu.com/s?wd=a&rsv_spt=1&issp=1&rsv_bp=0&ie=utf-8&tn=baiduhome_pg&inputT=1236）

  • 该正则表达式能够匹配端口号；

弱密码（以字母开头，长度在6~18之间，只能包含字母、数字和下划线）

$pattern = "/^[a-zA-Z]\w{5,17}$/"

强密码（必须包含大小写字母和数字，不能使用特殊字符，长度在8-10之间）

$pattern = "/^(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,10}$/"

汉字

$pattern = "/^[\x{4e00}-\x{9fa5}]+$/u";  // UTF-8字符集
$pattern = "/^[".chr(0xa1)."-".chr(0xff)."]+$/";  // GBK字符集
