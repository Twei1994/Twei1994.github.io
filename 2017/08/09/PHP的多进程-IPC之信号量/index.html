正文

PHP 实现进程通信（IPC：InterProcess Communication）的方式有以下几种：共享内存、信号量、消息队列、信号、管道和 socket。

这篇的内容是 PHP 通过信号量加上共享内存实现进程通信。使用信号量需要 PHP 安装 semaphore 扩展（在 congfigure 时使用 --enable-sysvsem 和 --enable-sysvshm 配置）。

信号量

首先要声明的是，单独通过信号量是不能实现进程间通信的，它要配合共享内存使用。在上一篇的最后，我们发现单独使用共享内存，由于无法控制进程读取共享内存的时间，导致多个进程会同时读取共享内存，不符合我们的要求。所以我们需要一个锁机制，来保证同一时间只有一个进程能获取到共享内存。信号量就是用于实现这个锁机制的。

信号量，英文是 semaphore，原义是旗语，信号灯，从这个方面更容易理解它作为锁的含义。

举个例子，我们今天看到的铁路大多是双线并行，但有的路段受山势、地形影响只有单条铁轨，必须保证同一时间只能有一列火车运行通过这些路段。早先铁路上就是用信号灯来管理的：没有火车经过时，信号等处于闲置状态，一旦有火车进入此路段，信号灯即变为在用状态，其他的火车经过时就需要等待，等待先前的火车驶出路段信号等变为闲置后，才能进入此路段，一旦又有火车进入，信号灯又变为繁忙……，以此来保障铁路运行的安全畅通。

信号量对共享内存的管理和信号灯对火车的管理是类似的。首先，生成一个信号量。当某个进程想要操作这个共享内存时，先看是不是能够获取到这个信号量，如果能，就占用它并操作共享内存（相当于给这个共享内存上锁），操作结束后，释放这个信号量；如果不能获取到这个信号量，说明别的进程正在使用它，就进行等待，直到可以获取到。

实例

修改上一篇博客的最后一个实例，增加信号量：

<?php
// 创建共享内存区域
$shm_key = ftok(__FILE__, 't');
$shm_id = shm_attach( $shm_key, 1024, 0655 );
const SHARE_KEY = 1;
$childList = [];

// 创建信号量
$sem_id = ftok(__FILE__,'s');
$signal = sem_get( $sem_id );

// 开3个进程 读写共享内存
for( $i = 0; $i < 3; $i++ ){

    $pid = pcntl_fork();
    if( $pid == -1 ){
        exit('fork fail!' . PHP_EOL);
    }else if( $pid == 0 ){

        // 获取信号量
        sem_acquire($signal);

        // 子进程从共享内存块中读取 写入值 +1 写回
        if ( shm_has_var($shm_id, SHARE_KEY) ){
            // 有值,加一
            $count = shm_get_var($shm_id, SHARE_KEY);
            $count ++;
            //模拟业务处理逻辑延迟
            $sec = rand( 1, 3 );
            sleep($sec);

            shm_put_var($shm_id, SHARE_KEY, $count);
        }else{
            // 无值,初始化
            $count = 0;
            //模拟业务处理逻辑延迟
            $sec = rand( 1, 3 );
            sleep($sec);

            shm_put_var($shm_id, SHARE_KEY, $count);
        }

        echo "child process " . getmypid() . " is writing ! now count is $count\n";
        // 用完释放
        sem_release($signal);
        exit( "child process " . getmypid() . " end!\n" );
    }else{
        $childList[$pid] = 1;
    }
}

// 等待所有子进程结束
while( !empty( $childList ) ){
    $childPid = pcntl_wait( $status );
    if ( $childPid > 0 ){
        unset( $childList[$childPid] );
    }
}

//父进程读取共享内存中的值
$count = shm_get_var($shm_id, SHARE_KEY);
echo "final count is " . $count . PHP_EOL;

// 从系统中移除内存共享
shm_remove($shm_id);
// 关闭和共享内存的连接
shm_detach($shm_id);
