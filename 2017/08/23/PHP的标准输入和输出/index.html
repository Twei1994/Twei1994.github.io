PHP的标准输入和输出

前言

写了这么久的 PHP，到现在连 PHP 的标准输入和输出都不知道，真是惭愧。直到昨天做笔试题的时候，遇到了这个要求，结果傻眼了。先不说算法实现，就连参数都获取不到，还谈什么 AC 呢。于是，笔试结束后，赶紧查了相关资料，学习下 PHP 的标准输入和输出。

正文

php://stdin php://stdout php://stderr

PHP 提供了 3 个输入输出流来处理输入，输出和错误设备，分别是 php://stdin，php://stdout 和 php://stderr。对它们的使用与使用文件类似（把它们 3 个看成 3 个文件即可），通过 fopen、fgets、fwrite、fread 和 fclose 来实现。

注： php://stdin 是只读的（”r”），php://stdout 和 php://stderr 是只写的（”w”）

实例

<?php
// 标准输入
$fp_in = fopen("php://stdin", "r");
$name = trim(fgets($fp_in));
echo $name;
fclose($fp_in);

// 标准输出
$msg = "Hello World";
$fp_out = fopen("php://stdout", "w");
fwrite($fp_out, $msg);
fclose($fp_out);

// 标准错误
$error = "Error!!!";
$fp_err = fopen("php://stderr", "w");
fwrite($fp_err, $error);
fclose($fp_err);

STDIN STDOUT STDERR

是不是感觉上面的方法有一点麻烦，每次都要打开和关闭相关文件句柄。PHP 在 CLI 模式中给我们提供了 3 个常量，分别代表了响应的文件句柄。

常量 含义
STDIN 标准的输入设备
STDOUT 标准的输出设备
STDERR 标准的错误设备

实例

还是上面的功能，通过这 3 个常量实现：

<?php
$name = trim(fgets(STDIN));
echo $name,"\n";

$msg = "Hello World!\n";
fwrite(STDOUT, $msg);

$err = "Error!!";
fwrite(STDERR, $err);

一个实例

面试中常见的问题：输入第一行表示点的个数，剩下的表示点的坐标。例如：

3
[1,2]
[3,4]
[1,4]

那么，我们怎么实现这个步骤呢？可以采用如下代码：

<?php
$num = trim(fgets(STDIN));
for ($i = 0; $i < $num; $i++) { 
    $points[] = trim(fgets(STDIN));
}

这样，$num 里面就是个数， $points 里面就是点的坐标（注意，为字符串，不是数组）。

可以使用这个函数将 “[1,2]” 类型的字符串转成数组：

function strToArr($str) {
    list($fir, $sec) = explode(",", $str);
    list( , $fir) = explode("[", $fir);
    list($sec, ) = explode("]", $sec);
    return array($fir, $sec);
}
