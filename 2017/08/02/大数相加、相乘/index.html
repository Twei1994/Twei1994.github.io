大数相加、相乘

题目描述

当一个数很大的时候，超过了 int 的范围，只能放在字符串里面。请实现两个字符串表示的数字的加法和乘法。

思路

整体的思路就是模仿我们的手算过程，一位一位的进行运算。

加法

例如要实现 234 + 7899，可以看做是 0234 + 7899，这样它们的位数就相同了。每次从后向前取一位，二者相加（同时要加上后一位的进位），如果大于 9，就会产生进位。所以我们要有个变量来保存进位值。并且要注意考虑最高位运算后的进位。

算法实现

function add($a, $b) {
    $len_a = strlen($a);
    $len_b = strlen($b);
    $res = '';
    $flag = 0;   // 进位标志
    for ($i = $len_a-1, $j = $len_b-1; $i >= 0 || $j >= 0; $i--, $j--) { 
        // 超出的位数用0表示
        $item_a = ($i >= 0)? $a[$i]: 0;
        $item_b = ($j >= 0)? $b[$j]: 0;
        // 求位数和
        $sum = (int)$item_a + (int)$item_b + $flag;
        // 是否进位
        if ($sum >= 10) {
            $flag = 1;  // 只可能为1
            $sum = $sum-10;
        }else{
            $flag = 0;
        }
        $res = (string)$sum.$res;
    }
    // 处理最高位进位
    if ($flag > 0) {$res = (string)$flag.$res;}
    return $res;    
}

乘法

乘法可以看成是多个一位数乘以多位数后相加得到。例如 124*59 = 4*59 + 2*59*10 + 1*59*100。最后乘的 10 的 n 次方，我们需要用补 0 的方式实现，因为随着位数的增加，10 的 n 次方也要用字符串表示。

这样就是两个循环，外层循环遍历第一个数的每一位，内层循环实现它与第二个数的乘法，然后补 0 后把它们加起来。

一位数乘以多位数的运算与加法运算类似，要考虑到进位。

算法实现

function product($a, $b) {
    $len_a = strlen($a);
    $len_b = strlen($b);
    $res = "0";
    for ($i = $len_a-1; $i >= 0; $i--) { 
        // 构造补的0
        $zero = '';
        for ($j = 0; $j < $len_a-1-$i; $j++) { 
            $zero .= '0';
        }
        // 每一位与第二个数相乘
        $flag = 0;
        $temp = '';
        for ($j = $len_b-1; $j >= 0; $j--) { 
            $pro = (int)$a[$i]*(int)$b[$j] + $flag;
            if ($pro >= 10) {
                $flag = floor($pro/10);   // 十位数为进位
                $pro = $pro%10;     // 个位数为该位结果
            }else{
                $flag = 0;
            }
            $temp = (string)$pro.$temp;
        }
        // 处理进位
        if ($flag > 0) {$temp = (string)$flag.$temp;}
        // 补零
        $temp .= $zero;
        // 将这次结果加给最终结果
        $res = add($res, $temp);
    }
    return $res;
}
