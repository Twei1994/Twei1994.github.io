PHP导出数据到Excel

前言

PHP导出数据到Excel中，是一个很常见的需求。之前知道通过PHPExcel库可以实现，但是如果数据不多或者没有样式要求时，没有必要使用拓展来实现。这篇文章来记录下除了使用PHPExcel之外，导出数据到Excel的方法。

正文

直接写入文件

直接新建一个文件，将数据写入。

示例代码：

<?php
$filename = "./test.xls";
$content = "aaa\t1232343423\tasdfasdf\n";
$fp = fopen($filename, "w");
for ($i = 0; $i < 5; $i++) { 
    fwrite($fp, $content);
}
fclose($fp);

这样的确可以生成一个Excel文件，但是打开文件时会有如下提示：

不过内容还是可以看到的，结果如下：

如果是生成.xlsx的Excel文件，则打开时提示如下错误：

无法打开文件，查看内容。

.xlsx是Microsoft Office EXCEL 2007使用的后缀名，是用新的基于XML的压缩文件格式取代了其目前专有的默认文件格式，在传统的文件名扩展名后面添加了字母x（即.docx取代.doc、.xlsx取代.xls，等等），使其占用空间更小。

xls格式的文件所有OFFICE版本都能打开，但.xlsx只有OFFICE2007或者更新的版本才能打开。

通过HTML中的<table>实现

示例代码：

<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<?php
$date = date("YmdHis");
header("Content-type:application/vnd.ms-excel");
header("Content-Disposition:filename=$date.xls");
header("Content-Type: application/vnd.ms-excel; charset=UTF-8");
?>
<table>
<thead>
    <th>序号</th>
    <th>姓名</th>
    <th>学号</th>
</thead>
<tbody>
    <tr>
        <!-- 写入内容 -->
        <td><?php echo "$id"?></td>
        <td><?php echo "$name"?></td>
        <td><?php echo "$student_id"?></td>
    </tr>
</tbody>

通过将生成的HTML页面下载下来并保存成.xls文件实现。

生成的Excel文件内容如下所示：

当写入的数字很大时（如纯数字的身份证号），会导致最后的几位变成0。可以在后面加上&nbsp来解决，如<td><?php echo "$student_id&nbsp"?></td>

不过，这样其实与直接写入.xls文件没有什么区别，在打开时也会提示“文件格式与拓展名不符”的错误，也不能生成.xlsx文件。

使用这个方法的好处是，可以生成带样式的表格。

生成带样式的表格

通过设置<table>的一些属性，可以生成带简单样式的表格，如背景色、合并单元格、边框等。

实例代码：

<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<?php
$date = date("YmdHis");
header("Content-type:application/vnd.ms-excel");
header("Content-Disposition:filename=$date.xls");
header("Content-Type: application/vnd.ms-excel; charset=UTF-8");
?>
<table border=1>
<thead>
    <th colspan="3" align="center">
        <h2>成绩单</h2>
    </th>
</thead>
<tbody>
    <tr>
        <td align="center">序号</td>
        <td align="center">姓名</td>
        <td align="center">学号</td>
    </tr>
    <tr>
        <td style="background-color: #00CC00;" align="center"><?php echo "$id"?></td>
        <td style="background-color: #00adee;" align="center"><?php echo "$name"?></td>
        <td style="background-color: #00CC00;" align="center"><?php echo "$student_id&nbsp"?></td>
    </tr>
</tbody>

通过border属性设置表格边框，通过backgroud-color属性设置背景颜色，通过align属性设置文字位置，通过colspan和rowspan来合并单元格。

总结

  1. 通过直接写文件和HTML的<table>都可以生成Excel；
  2. 但是只能生成.xls文件，不能生成.xlsx文件；
  3. 通过<table>的属性可以设置一些简单的属性；
