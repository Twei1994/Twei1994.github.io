####总结：

  1. 整体上感觉R和MATLAB很像（但是不以;结尾）

  2. 查看帮助：

    函数：help(FunctionName)或者？FunctionName
    包：help(package = “PackageName”)

  3. RStudio快捷键：

    ctrl+l 清空控制台
    ESC 中断当前计算
    ctrl+shift+c 注释，取消注释
    ctrl+Enter 运行该行代码
    ctrl+shift+Enter 运行全部代码

  4. 包（package）的安装：task view中找；packages页面中

    下载到本地，程序包->从本地安装->加载程序包
    在线下载安装，程序包->安装程序包->选择镜像->选择包->加载程序包

  5. 清除变量:

    rm(list = ls()) #清除所有变量
    rm(a) #清除变量a

笔记：

一、数据对象与运算

  1. 数据对象：Object，变量，数组，字符串，函数或者其它的结构，R中的所有东西

  2. 数据类型：

    a.数值型（numeric）：整数型+双精度型（默认）
    b.逻辑型（logical）：T（TRUE）或F（FALSE）
    c.字符型（character）：””或’’之间
    d.复数型（complex）：a+bi，复数
    e.原味型（raw）:二进制形式
    f.缺省值（missing value）：NA（not available），统计缺失

  3. 判断和转化数据类型：is.xxx(); as.xxx()

  4. 数据对象构造：
    a.向量（vector）：用x = c()来创建
    b.矩阵（matrix）：用matrix()来创建
    c.数组（array）：用array()来创建
    d.列表（list）：内部元素可以有不同的数据类型，用list()来创建
    e.数据框（data frames）：一列表示一个变量，一行表示一个观察值，用data.frame()来创建，类似与一张表
    f.因子（factor）：分类数据，如性别包含“男”，“女”两个因子，用factor()（无序）或ordered()来生成（有序）

  5. apply(X,MARGIN,FUN,…):X:处理的数据；MARGIN，函数作用范围，1表示对每一行，2表示对每一列；FUN：采用的函数

  6. array(data=NA,dim=length(data),dimnames=NULL)

二、数据的录入和编辑

  1. c()函数，把值连成向量或列表

  2. scan()函数，键盘输入数据，也可以读外部文件

  3. read.table()函数，读取外部文件txt文件，文件在工作路径下（或写好文件路径）

  4. url()读取网上的数据，read.table(file = url(“http:..”))

  5. 读入其他格式的数据库：先载入foreign包,library(foreign)

    a.SAS数据库，只能读取XPORT格式，命令：read.xport()
    b.SPSS数据库，read.spss()
    c.Epi info数据库，read.epiinfo()
    d.Excel表，CSV格式，read.csv()

  6. 编辑数据

    data.entry(xx=c(NA))
    data.entry(z) #z是已存在的数据
    edit()和fix()

  7. 导出数据

    write.table(student,”student.txt”)
    write.csv(student,”student.csv”)
