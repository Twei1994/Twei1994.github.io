MarkdownPad 2 支持表格

前言

因为是在Windows下使用Hexo来写博客，所以就选择了MarkdownPad 2作为编辑器。但是在使用的时候发现默认的设置并不支持表格输入，于是就找了下解决方法。

解决方法

原因

导致不能输入表格的原因是由于默认的MarkdownPad 2处理器是Markdown，这个处理器是不支持表格的。

解决方法

既然知道了原因，那么解决方案就是调整MarkdownPad 2的处理器了，方法有两种：

  1. 点击 工具 >> 选项 >> Markdown >> Markdown 处理器，改成Markdown（扩展）即可。

    英文环境下是 Tools >> Options >> Markdown >> Markdown Processor，改成Markdown(Extra)。

    顺便说一下，修改软件语言的操作是：Tools >> Options >> Editor >> Languages。

  2. 点击界面左下角的M标签，选择处理器为 Markdown（扩展）即可。

Markdown表格语法

| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned |  foo  |
| col 2 is      | centered      |  bar  |
| col 1 is      | default       |  baz  |
Tables Are Cool
col 3 is right-aligned foo
col 2 is centered bar
col 1 is default baz

语法说明

  • |、-、:之间的多余空格会被忽略，不影响布局。
  • 默认标题栏居中对齐，内容居左对齐。
  • -:表示内容和标题栏居右对齐，:-表示内容和标题栏居左对齐，:-:表示内容和标题栏居中对齐。
  • 内容和|之间的多余空格会被忽略，每行第一个|和最后一个|可以省略，-的数量至少有一个。
