ThinkPHP中验证码的用法

前言

本篇博客主要说明ThinkPHP中验证码类的使用方法，并不涉及相关实现原理。

基本实现

首先，我们看一下ThinkPHP官方手册中对于验证码使用的介绍。

生成验证码

ThinkPHP中的Think\Verify类是用于生成验证码和进行验证的类。

当我们想要生成验证码时，只要实例化一个该类的对象，并通过entry()函数将生成的验证
码图片输出。

$Verify = new \Think\Verify();
$Verify = entry();

验证码检测

可以用Think\Verify类的check方法检测验证码的输入是否正确，下面是手册提供的一个封装好的验证码检测的函数：

// 检测输入的验证码是否正确，$code为用户输入的验证码字符串
function check_verify($code, $id = ''){
    $verify = new \Think\Verify();
    return $verify -> check($code,$id);
}

验证码参数设置

直接通过实例化Verify类生成的验证码图片的参数是ThinkPHP默认的。样子是这样的：

我们可以调整不同的参数从而获得我们想要的效果。这些参数包括：

参数 描述
expire 验证码的有效期（秒）
useImgBg 是否使用背景图片 默认为false
fontSize 验证码字体大小（像素） 默认为25
useCurve 是否使用混淆曲线 默认为true
useNoise 是否添加杂点 默认为true
imageW 验证码宽度 设置为0为自动计算
imageH 验证码高度 设置为0为自动计算
length 验证码位数
fontttf 指定验证码字体 默认为随机获取
useZh 是否使用中文验证码
bg 验证码背景颜色 rgb数组设置，例如 array(243, 251, 254)
seKey 验证码的加密密钥
codeSet 证码字符集合
zhSet 验证码字符集合（中文）

参数的设置有两种方式，可以在实例化时将参数对应的数组传入：

$congfig = array(
    'fontSize' => 30,    //字体大小
    'length'   => 3,    //文字的数量
    'useNoise' => false,//不使用背景杂点 
);
$Verify = new \Think\Verify($config);
$Verify -> entry();

另外一种方式是动态设置的方式：

$Verify =     new \Think\Verify();
$Verify->fontSize = 30;
$Verify->length   = 3;
$Verify->useNoise = false;
$Verify->entry();

public function verify(){

假设控制器的名称是Login，操作方法是verify()。

公共函数

// 验证码检测函数
function check_verify($code, $id = ''){
    $verify = new \Think\Verify();
    return $verify -> check($code,$id);
}

这个函数可以放在\Common\founction.php中，在验证用户输入验证码的时候指教调用就可以。

检查用户输入是否正确

用户输入后，点击登录按钮，执行check操作方法，同样是在Login控制器下。

public function check(){
    //首先获得用户输入
    $code = I('POST.verify');
    if(! check_verify($code)){
        $this -> error('验证码输入错误！');
    }else{
        $this -> success('验证成功！');
    }
}
