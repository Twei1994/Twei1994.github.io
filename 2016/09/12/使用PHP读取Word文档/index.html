使用PHP读取Word文档

前言

因为在上线前，需要向网站的数据库存入相关信息，进行内测。但是由于用户提供的资料是Word版的，并且量比较大，所以研究了下用PHP打开Word文档，读取内容并存入MySQL中。

准备工作

COM（Component Object Model）组件对象模型，是一种跨应用和语言共享二进制代码的方法。COM可以作为DLL被本机程序载入也可以通过DCOM被远程进程调用。

因为要使用COM组件，所以要在PHP的配置文件php.ini中打开COM组件。如果未开启该组件，会报错，未找到COM类。

Fatal error: Class ‘COM’ not found

不同版本的PHP设置方法不同。所以，首先要查看PHP的版本。

另外，我的操作系统是Windows，PHP版本为5.4.6。

查看PHP版本

使用phpinfo来查看本机安装的php的基本信息以及相关组件安装情况。

<?php
phpinfo();
?>

打开COM组件

如果PHP版本在4.2.0到4.2.3，可以使用w32api_register_function函数调用外部的DLL，前提是需要在php.ini中打开扩展的php_w32api.dll。
也就是说，去掉extension=php_w32api.dll前的注释。

如果PHP版本在5.4.5之后，com/dotnet模块已经成了单独的扩展，所以需要在php.ini中配置extension=php_com_dotnet.dll。
也就是说，在php.ini最后，添加extension=php_com_dotnet.dll即可。

开启COM组件后，再次通过phpinfo查看，可以看到COM/DOTNET组件已经可用。
