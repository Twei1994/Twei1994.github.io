Matlab神经网络学习--反馈神经网络（介绍）

介绍

无论是BP神经网络还是径向基神经网络，都属于前向神经网络（前馈型神经网络）。在这张网络中，数据正向流动，没有反馈连接。而反馈神经网络的输出除了与当前输入和网络权值有关以外，还与网络之前的输入有关。

Hopfield网络

Hopfield网络是一种单层反馈神经网络，又被称作联想记忆网络，分为离散型（DHNN）和连续型（CHNN）两种。它的结构如下图所示：

输出值用1和-1表示，自身的输出不会再作为自身的输入，也就是不包含自反馈。仅考虑中间的神经元节点，可以将Hopfield网络用网状结构表示：

Hopfield网络的训练过程就是使能量函数（Lyapunov函数）减小的过程，最终达到稳定状态，称为吸引子（也就是每一个类别）。能够演化至某吸引子的状态集合就是该吸引子的吸引域。网络的实现分为两个阶段：记忆阶段和联想阶段。记忆阶段类似于训练，通过外部输入，调整权值，使得网络具有若干个吸引子。联想阶段类似于测试，对于给定输入模式，系统经过一定的演化过程，最终稳定在某个吸引子。

Elman神经网络

Elman神经网络是一种带有反馈的两层BP网络结构，其反馈连接是从隐含层的输出到其输入端。这种反馈方式使得Elman网络能够探测和识别时变模式。

其隐含层又称为反馈层，神经元的传递函数为sigmoid，输出层为线性层，传递函数为purelin。这种特殊的两层网络可以任意精度逼近任意函数，唯一的要求是其隐含层必须具有足够的神经元数目。隐含层神经元数越多，则逼近复杂函数的精度就越高。

Elman网络和传统的两层神经网络的不同之处在于其第一层具有反馈连接，它可以存储前一次的值，并应用到本次计算中。反馈状态不同，则输出结果不同。因为网络可以存储信息，所以它不但可以存储空间模式，也可以学习时间模式。

常用函数

newhop—生成一个离散hopfield网络

net = newhop(T)

T是一个R*Q的矩阵，包含Q个长度为R的向量，作为Hopfield网络的记忆模式向量。取值为1或-1。Hopfield网络除了输入和输出层之外只有一层，网络的权值函数为dotprod函数，网络输入函数为netsum，传输函数使用satlins函数。

elmannet—

net = elmannet(layerdelays, hiddenSizes, trainFcn)
  • layerdelays: 表示网络层延迟的行向量，可取的值为0或正数，默认值为1:2
  • hiddenSizes: 隐含层的大小，是一个行向量，默认值为10
  • trainFcn: 表示训练函数的字符串，默认值为‘trainlm’

仿真实验

实验1

描述：用newhop创建一个联想记忆网络，并利用该网络记住向量[-1，-1,1]和[1，-1,1]。

T = [-1 -1 1; 1 -1 1]'        % 吸引子为向量[-1,-1,1]和[1,-1,1]
% T =
%     -1     1
%     -1    -1
%      1     1

net = newhop(T);            % 使用newhop创建联想记忆网络
Ai = T;
[Y,Pf,Af] = net(2,[],Ai)    % 使用原输入进行仿真
% Y =
%     -1     1
%     -1    -1
%      1     1
% Pf =
%      []
% Af =
%     -1     1
%     -1    -1
%      1     1

Ai = {[-0.9; -0.8; 0.7]};    % 使用不同的数据进行仿真
[Y,Pf,Af] = net({1 5},{},Ai);
Y{1}
% ans =
%     -1
%     -1
%      1

给定两个吸引子作为输入，系统返回了正确的结果，给定一个接近吸引子的向量作为输入，网络经过迭代返回与其距离最近的吸引子。

实验2，用elmannet建立Elman网络用于预测时序数据

描述：给出7天的空调负荷数据，用前6天的数据去预测第7天的数据

数据如下：

% 清理
close all;clear;clc
% 原始数据
data =[0.4413    0.4707    0.6953    0.8133;...
       0.4379    0.4677    0.6981    0.8002;...
       0.4517    0.4725    0.7006    0.8201;...
       0.4557    0.4790    0.7019    0.8211;...
       0.4601    0.4811    0.7101    0.8298;...
       0.4612    0.4845    0.7188    0.8312;...
       0.4615    0.4891    0.7201    0.8330];
rng('default')
rng(2);

for i=1:4      % 对于每一列，以前3天数据为特征，第4天数据为标签训练
    P=[data(1:3,i),data(2:4,i),data(3:5,i)];
    T=[data(4,i),data(5,i),data(6,i)];

    net=elmannet(1:3,20);               % 创建Elman网络
    net=init(net);                      % 初始化
    net=train(net,P,T);                 % 训练

                                     % 测试
    test_P=data(4:6,i);
    y(i)=sim(net,test_P);               % 仿真
end
fprintf('真实值:\n');
disp(data(7,:));
fprintf('预测值:\n');
disp(y);
fprintf('误差:\n');
disp((y-data(7,:))./y);
fprintf('平均误差mse：\n');
disp(mean(abs(y-data(7,:))))

执行结果：

真实值:
    0.4615    0.4891    0.7201    0.8330

预测值:
    0.4630    0.4839    0.7208    0.8336

误差:
    0.0032   -0.0107    0.0009    0.0007

平均误差mse：
    0.0020

elmannet中第二个参数用于控制神经元个数，适当增大该参数可以确保更好的误差性能。
