Matlab神经网络学习--反馈神经网络（应用）

介绍

离散Hopfield网络适用于联想记忆，Elman神经网络适合解决时间序列问题。

实验

实验1，Hopfield网络，坐标点的联想记忆问题

描述：在二维平面上定义两个平衡点，[1,-1]和[-1,1]，使所有的输入向量经过迭代最后都收敛到这两个点。

网络的结构如下：

手算

采用串行运行方式，权值矩阵必须为对称阵，且主对角线元素必须为0。采用外积法进行训练。

%% 清理
close all;clear;clc

%% 
disp('吸引子坐标为:');
c1=[-1,1]                        % 第一个平衡点
c2=[1,-1]                        % 第二个平衡点

% 计算权值矩阵
w=zeros(2,2);
for i=1:2
   for j=1:2
      if (i~=j)
         w(i,j)=1/2*(c1(i)*c1(j) + c2(i)*c2(j));   % 外积法
      end
   end
end

% 阈值向量
b=[0,0];

disp('权值矩阵为');
w
disp('阈值向量为');
b

% 网络初始值
rng(0);
y=rand(1,2)*2-1;
y(y>0)=1;
y(y<=0)=-1;      % 产生随机数并二值化作为初始状态

% 循环迭代
disp('开始迭代');
while 1
    % 保存上一次的网络状态值
    disp('网络状态值:');
    tmp = y

    % 更新第一个神经元
    y_new1 = y * w(:,1) + b(1);
    fprintf('第一个神经元由 %d 更新为 %d \n', y(1), y_new1);
    y=[y_new1, y(2)];

    % 更新第二个神经元
    y_new2 = y * w(:,2) + b(2);
    fprintf('第二个神经元由 %d 更新为 %d \n', y(2), y_new2);
    y=[y(1), y_new2];

    % 如果状态值不变，则结束迭代
    if (tmp == y)
        break;
    end

end

使用工具箱

% 定义吸引子
T = [-1,  1;...
     1,  -1]

 % 创建hopfield网络
 net=newhop(T);

 % 用原平衡位置的坐标作为输入进行仿真
 Y = sim(net,2,[],T);
 fprintf('输入平衡中心得出的结果：\n');
 disp(Y);

 % 用新的值作为输入，用3个新数据进行测试
 rng(0);
 N=3;
 for i=1:N
     y=rand(1,2)*2-1;
     y(y>0) = 1;
     y(y<0) = -1;
     [Y,a,b]=sim(net,{1,5},[],y');
     b = b{1};
     if (sum(abs(b))<1.0e-1)
         b=[0,0]';
     end
     fprintf('第 %d 组测试数据: ',i);
     disp(y);
     fprintf('网络输出:         ');
     disp(b');
end

这里，部分输入，如[1,1]，没有收敛到平衡中心，而是在0值附近进入了振荡或者混沌状态，所以设置了一个阈值1.0e-1来检测这个状态，并将其对应的输出赋值为[0,0]。

Hopfield网络的问题主要就是稳定性和伪状态，并不是所有的输入都能正确记忆出正确的平衡点。

实验2，用Elman网络预测股票

描述：Elman网络适合处理时间序列问题，原始资料是某股票280天的数据，采用前140天作为训练样本，其中每连续5天作为输入，第6天作为期望输出。

股票的数据放在这里，使用load加载后，绘图，可以看到股票的情况如下：

代码如下：

%% 清理
close all;clear;clc

%% 加载数据
load stock1

% 归一化处理
mi=min(stock1);
ma=max(stock1);
stock1=(stock1-mi)/(ma-mi);

% 划分训练数据与测试数据：前140个维训练样本，后140个维测试样本
traindata = stock1(1:140);
testdata = stock1(141:280);

%% 训练
% 输入
P=[];
for i=1:140-5
    P=[P;traindata(i:i+4)];
end
P=P';

% 期望输出
T=[traindata(6:140)];

% 创建Elman网络
net=elmannet;
%  开始训练
% 设置迭代次数
net.trainParam.epochs=1000;
% 初始化
net=init(net); 
net=train(net,P,T);

%% 使用训练数据测试一次
y=sim(net,P);
error=y-T;
mse(error);

% 从归一化中恢复
stock1 = stock1*(ma-mi) + mi;    
y = y*(ma-mi) + mi;

plot(6:140,stock1(6:140),'b-',6:140,y,'r-');
title('使用原始数据测试');
legend('真实值','测试结果');

%% 用后140期数据做测试
% 输入
Pt=[];
for i=1:135
    Pt=[Pt;testdata(i:i+4)];
end
Pt=Pt';
% 测试
Yt=sim(net,Pt); 

%根据归一化公式将预测数据还原成股票价格
YYt=Yt*(ma-mi)+mi;

%目标数据-预测数据
figure
plot(146:280, stock1(146:280), 'r',146:280, YYt, 'b');
legend('真实值', '测试结果');
title('股价预测测试');
