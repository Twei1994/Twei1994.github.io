Matlab神经网络学习--自组织竞争神经网络（介绍）

介绍

自组织竞争神经网络采用了与前向神经网络完全不同的思路。它采用了竞争学习的思想，网络的输出神经元之间相互竞争，同一时刻只有一个输出神经元获胜，称为“胜者全得”神经元或获胜神经元。这种神经网络的仿生学基础是神经元之间的侧抑制现象。

采用竞争学习的规则即可构成最简单的竞争神经网络，在此基础上，还发展出了各种自组织网络。与竞争网络相比，自组织网络除了能学习输入样本的分布外，还能够识别输入向量的拓扑结构。

竞争神经网络

竞争神经网络只有两层，输入层和输出层（核心层）。最初，输入层和核心层之间的权值是随机给定的。因此，每个核心层神经元获胜的概率相同，以输入和权值向量的距离作为依据，得到一个兴奋最强的神经元。该神经元的权值会被加强，其他的不变。（整体上类似与K-means聚类）

权重的更新采用Kohonen规则，为了解决某些神经元由于初始阈值不合理始终无法获胜的问题，引入了阈值学习规则（Bias Learning Rule）。

自组织特征映射网络

自组织特征映射网络（Self-Organizing Feature Maps, SOFM）又称为自组织映射网络（SOM）。它是一种竞争性神经网络，同时引入了自组织特性。除了能学习输入样本的分布外，还能识别输入向量的拓扑结构。所以对于识别结果并不是单个神经元在起作用，而是需要多个神经元协同作用完成。对映在网络的结构上，输出层是具有拓扑结构的网络，彼此之间通过权值相连。默认的拓扑结构是六边形的。

SOM的学习过程分成了两个步骤，一个是排序阶段（order），另一个是调整阶段（tuning）。排序阶段的学习率和更新的领域范围比较大，在调整阶段这两个值就减小了，保证学习的稳定性。更新的规则还是Kohonen。

学习矢量量化网络

不管是竞争神经网络还是SOM，都是无监督的聚类。当样本的类别已知时，可以将自组织竞争和有监督学习相结合，这就是学习矢量量化网络（Learning Vector Quantization, LVQ）。

相当于在SOM的基础上又增加了一个线性层，线性层根据类别标签对竞争层的结果进行分类，更新权值。

常用函数

gridtop–网络拓扑函数

pos = girdtop(dim1, dim2, ..., dimN)

用于创建SOM网络中输出层的网络拓扑结构。输入参数个数表示拓扑结构的维数，参数大小表示拓扑结构的形状大小。

类似的函数还有hextop，randtop，tritop。

dist、boxdist、linkdist、mandist—距离函数

这几个函数是用于求取两个向量之间的距离的。

dist: 欧式距离

boxdist: 向量各分量绝对差的最大值

linkdist: 是SOM默认的距离函数

mandist: 曼哈顿距离，也叫出租车几何距离

competlayer—竞争网络函数

net = competlaryer(numClasses, kohonenLR, conscienceLR)
  • numClasses: 分类的类别数
  • kohonenLR: 学习率
  • conscienceLR: “良心”学习率，阈值学习规则的学习率

selforgmap—自组织映射网络函数

net = selforgmap(dimensions,coverSteps,initNeighbor,topologyFcn,distanceFcn)
  • dimensions: 表示拓扑结构的行向量，如[8,8]表示8*8的二维结构
  • coverSteps: 训练次数
  • initNeighbor: 领域大小的初始值
  • topologyFcn: 拓扑函数的字符串，可选值为“hextop”、“gridtop”、“randtop”等
  • distanceFcn: 距离函数的字符串，可选值为“dist”、“boxdist”、“linkdist”、“mandist”等

lvqnet—矢量量化网络函数

net = lvqnet(hiddenSize, lvqLR, lvqLF)
  • hiddenSize: 竞争层神经元节点个数
  • lvqLR: 学习率
  • lvqLF: 学习函数，分为“learnlv1”和“learnlv2”

实验1，创建网格拓扑结构

pos = gridtop(8,5);        % 创建网格
% pos = hextop(8, 5)    % 创建六边形拓扑结构
pos                     % 神经元的坐标

net = selforgmap([8 5],'topologyFcn','gridtop');
% net = selforgmap([8 5],'topologyFcn','hextop');
plotsomtop(net)            % 显示网络

selforgmap函数用于创建一个自组织映射网络，其默认拓扑结构为六边形。plotsomtop函数可以显示一个自组织网络的结构。

实验2，competlayer竞争网络函数

描述：iris_dataset是MATLAB自带的数据集，包含了150个样本，每个样本有4个特征，总共有3个类型。通过竞争神经网络分类。 

inputs = iris_dataset;            % 载入数据
net = competlayer(3);            % 创建竞争网络
net = train(net,inputs);        % 训练
outputs = net(inputs);            % 分类
classes = vec2ind(outputs)        % 格式转换
c=hist(classes,3)                % 每个类别的数量

实验3，selforgmap自组织映射网络聚类

描述：对MATLAB自带的数据集simplecluster_dataset进行聚类，第一次选取88个中心点，第二次选取23个中心点。

x = simplecluster_dataset;
plot(x(1,:),x(2,:),'o')
set(gcf,'color','w')
title('原始数据')

net = selforgmap([8 8]);        % 创建自组织映射网络
net = train(net,x);                % 训练
y = net(x);
classes = vec2ind(y);
hist(classes,64)                % 显示聚类结果
set(gcf,'color','w')
title('聚类结果')
xlabel('类别')
ylabel('类别包含的样本数量')
plotsomhits(net,x)          % 显示每个类别的个数
plotsompos(net,x)           % 显示类别中心点的位置

net = selforgmap([2,3]);
net = train(net,x);
y = net(x);
classes = vec2ind(y);

实验4，lvqnet矢量量化网络函数

描述：有监督的分类问题，对iris数据集进行分类

[x,t] = iris_dataset;        % 加载数据，x为输入样本，t为期望输出
rng(0)

ri=randperm(150);           % 划分训练与测试集
x1=x(:,ri(1:50));
t1=t(:,ri(1:50));
x2=x(:,ri(51:150));
t2=t(:,ri(51:150));
net = lvqnet(20);            % 创建网络进行训练
net = train(net,x1,t1);
y = net(x2);                % 测试
yy=vec2ind(y);
ty=vec2ind(t2);
sum(yy==ty)/length(yy)

输出值：

ans = 0.9100

lvqnet采用有监督学习，所以这里将数据分成了两份，50份用于训练，100份用于测试。训练的时间比较长，最终结果是正确率达到了91%。
