Matlab神经网络学习--BP神经网络（介绍）

介绍

BP神经网络介绍

线性神经网络只能解决线性可分的问题，对线性不可分的情况必须要采取迂回的方法才能实现，这是由其单层的网络结构决定的。BP神经网络在输入和输出层之间加入了隐含层，具备处理线性不可分的问题。但是，很难找到合适的算法来进行训练。直到20世纪80年代，才出现了误差反向传播算法（Error Back Propagation, BP），解决了多层网络不好训练的问题。

它具有以下几个特点：

  1. 网络是由多层构成，层与层之间采用全连接的方式，而同一层之间的神经元无连接；
  2. BP网络采用的传递函数必须是可微的，通常采用Sigmoid函数作为传递函数。常用的有两种，一种是Log-Sigmoid函数，另一种是Tan-Sigmod函数。
  3. 采用误差反向传播算法（Back-pripagation Algorithm）进行学习，也就是在训练网络权值时，从后向前，从输出层出发，逐层向前修正网络的连接权值。

BP网络的学习算法

与线性网络的学习算法类似，标准BP网络也采用了LMS算法（本质上就是最速下降法，或者梯度下降法），也就是沿着梯度相反的方向来前进。权重的更新公式可以总结成这样的：

权重的调整值 = 学习率 * 局部梯度 * 上一层的输出信号

当前层的局部梯度是后一层局部梯度的加权和，所以只能从后向前进行计算。

另外还有一些对于最速下降BP算法的改进，例如动量BP法，学习率可变的BP算法，拟牛顿法，LM（Levenberg-Marquardt）算法（较常用）。

常用函数

feedforwardnet–创建一个BP网络

net = feedforwardnet(hiddenSizes, trainFcn)
  • hiddenSizes: 隐含层神经元节点个数，如果有多个隐含层，则为一个行向量；
  • trainFcn: 训练函数，默认为“trainlm”

feedforwardnet函数并没有确定输入和输出层向量的维数，这些结构的确定是通过train函数完成的。

仿真实验

描述：用feedforwardnet建立网络，拟合一段曲线，并查看性能。

[x,t] = simplefit_dataset;        % MATLAB自带数据，x、t均为1*94向量
net = feedforwardnet;            % 创建前向网络
% view(net)                        % 用于查看网络结构
net = train(net,x,t);            % 训练，确定输入输出向量的维度
% view(net)
y = net(x);
perf = perform(net,y,t)            % 计算误差性能
plot(x, t, 'o-');
hold on;
plot(x, y, '*-');

实验结果：perf = 1.4639e-04
