Matlab神经网络学习--径向基函数网络（应用二）

介绍

广义回归神经网络（GRNN）以径向基网络为基础，相比较而言，更适合解决曲线拟合问题，速度更快（原因在于GRNN不需要训练权值）。

概率神经网络（PNN）适用于解决模式分类问题。

实验

实验1，GRNN拟合曲线

手写grnn_net函数

function y = grnn_net(p,t,x,spread)
% grnn_net.m
% p：R*Q矩阵，包含Q个长度为R的输入
% t：1*Q行向量，对应于p的期望输出
% x：测试数据
% spread：平滑因子，可选，缺省值为0.5

    if ~exist('spread','var')
        spread=0.5;            % spread默认值为0.5
    end

    [R,Q]=size(p);
    [R,S]=size(x);

    %% 径向基层的输出
    yr = zeros(Q,S);
    for i=1:S
        for j=1:Q
            v = norm(x(:,i) - p(:,j));
            yr(j,i) = exp(-v.^2/(2*spread.^2));
        end
    end

    %% 加和层的输出
    ya = zeros(2,S);
    for i=1:S
        ya(1,i) = sum(t .* yr(:,i)',2);
        ya(2,i) = sum(yr(:,i));
    end

    %% 输出层的结果
    y = ya(1,:)./(eps+ya(2,:));

end

调用grnn_net函数进行曲线拟合：

%% 清理
close all;clear all;clc

%% 训练数据
x=-9:8;
y=[129,-32,-118,-138,-125,-97,-55,-23,-4,...
    2,1,-31,-72,-121,-142,-174,-155,-77];
P=x;
T=y;

%% 设计网络与测试
xx=-9:.2:8;
yy = grnn_net(P,T,xx);        % spread值选用默认值0.5

%% 显示
plot(x,y,'o') 
hold on;
plot(xx,yy)
hold off

结果图如下：

取不同的spread值，会产生不同的拟合效果。spread值越大，拟合越平滑（容易欠拟合），值越小，拟合越精细（容易过拟合）。

实验2，PNN坐标点分类

首先产生三类分类的样本点用于学习：

rng(2);
a = rand(14,2)*10;          % 训练点数据
p = ceil(a)'
tc = [3, 1, 1, 2, 1, 3, 2, 3, 2, 3, 3, 2, 2, 3];
plot(p(1,tc == 1), p(2,tc == 1), 'ro');
hold on;
plot(p(1,tc == 2), p(2,tc == 2), 'b*');
plot(p(1,tc == 3), p(2,tc == 3), 'k+');
axis([0,11,0,11]);
legend('第一类','第二类','第三类');
title('待分类数据')

根据PNN的定义写出pnn_net函数：

function clas = pnn_net(p,t,x,sigma)
% pnn_net.m
% p为训练输入,R*Q，Q个长度为R的向量
% t为训练输出,1*Q行向量，取值从1~C，表示C个类别，C<=Q
% x为测试输入,R*S矩阵，S个长度为R的列向量

if ~exist('sigma','var')
    sigma=0.1;
end

% 数据归一化
MAX = max(p(:));
p=p/MAX;
x=x/MAX;

% 向量长度M，向量个数N
[R,Q]=size(p);
[R,S]=size(x);

% 计算径向基层的输出,y(i,j)是第j个测试向量与第i个神经元的输出
y=zeros(Q,S);
for i=1:S
    for j=1:Q
        v = norm((x(:,i) - p(:,j)));
        y(j,i) = exp(-v^2/(2*sigma^2));
    end
end

% 相加层
% 共有C个类别
C = length(unique(t));
% 相加层输出
vc=zeros(C,S);
for i=1:C
    for j=1:S
        vc(i,j) = mean(y(t==i,j));
    end
end

% 输出层
yout=zeros(C,S);
for i=1:S
    [~,index] = max(vc(:,i));
    yout(index,i) = 1;
end

clas=vec2ind(yout);

end

对输入的样本数据进行测试：

%% 清理
close all
clear,clc

%% 定义数据
rng(2);
a=rand(14,2)*10;                    % 训练数据点

tc=[3,1,1,2,1,3,2,3,2,3,3,2,2,3];        % 类别
disp(tc);

%% 用训练数据测试

disp('测试结果：');
disp(y);

可见对于训练数据分类全部正确，为了达到更好的测试效果，用0~11区间内的所有网格点进行测试，并绘制结果图：

%% 清理
close all;clear all;clc

%% 定义数据
rng(2);
a=rand(14,2)*10;                    % 训练数据点
p=ceil(a)'
tc=[3,1,1,2,1,3,2,3,2,3,3,2,2,3];        % 类别

%% 生成测试数据
x=0:.4:11;
N=length(x);
for i=1:N
    for j=1:N
        xx(1,(i-1)*N+j) = x(i);
        xx(2,(i-1)*N+j) = x(j);
    end
end

%% 测试
y = pnn_net(p,tc,xx,1);

%% 显示     
plot(xx(1,y==1),xx(2,y==1),'ro');
hold on;
plot(xx(1,y==2),xx(2,y==2),'b*');
plot(xx(1,y==3),xx(2,y==3),'k+');
plot(p(1,tc==1),p(2,tc==1),'ro','LineWidth',3);
plot(p(1,tc==2),p(2,tc==2),'b*','LineWidth',3);
plot(p(1,tc==3),p(2,tc==3),'k+','LineWidth',3);
axis([0,11,0,11])
legend('第一类','第二类','第三类');
title('分类结果');
