Matlab神经网络学习--径向基函数网络（应用）

介绍

严格径向基网络不存在误差，它的目的在于求取权值，求取步骤如下：

  1. 确定每个隐含层节点的中心点。常见的有4种：随机选取固定中心、自组织选取（K-means聚类）、有监督选取中心、正交最小二乘法；
  2. 计算输入向量和每个中心点的2-范数；
  3. 带入径向基函数（通常采用高斯函数）中，并增加偏置值；
  4. 根据期望输出和径向基矩阵求出权值矩阵。

实验

实验1，异或问题

描述：用径向基网络学习异或逻辑

手算法

手算法基本就是按照本页开头的步骤来计算权值。

%% 清理
clear all;close all;clc

%% 输入
% 输入向量
x=[0,0;0,1;1,1;1,0]

% 隐含节点的中心
t=[0,1;0,1]              % 选取中心点

%% 计算网络
% 计算输入向量到中心的距离
z=dist(x,t)
%z =
%          0     1.4142
%    1.0000    1.0000
%    1.4142         0
%    1.0000    1.0000

% 将算得的距离输入到径向基函数中
G=radbas(z)                % 高斯径向基函数
%G =
%    1.0000    0.1353
%    0.3679    0.3679
%    0.1353    1.0000
%    0.3679    0.3679

% 加上偏置
G=[G,ones(4,1)]
%G =
%    1.0000    0.1353    1.0000
%    0.3679    0.3679    1.0000
%    0.1353    1.0000    1.0000
%    0.3679    0.3679    1.0000    

% 期望输出
d=[0,1,0,1]'

% 求权值向量
w=inv(G.'*G)*G.'*d
%w =
   %    -2.5027
%    -2.5027
%    2.8413

%% 计算实际输出
Y=G*w
%Y =
%    -0.0000
%    1.0000
%    -0.0000
%    1.0000

可见，通过计算出来权值，代入训练数据，可以得到期望的输出。

观察矩阵G，忽略第3列偏置。第一行是第一个输入向量在隐含层的输出，其余的也类似。

1.0000 0.1353 1.0000

0.3679 0.3679 1.0000

0.1353 1.0000 1.0000

0.3679 0.3679 1.0000

在坐标上绘制出来，可以得到：

从中可以看到，RBF网络的隐含层经过高斯函数的运算，将原向量空间中的4个点（线性不可分）映射为隐含层空间中的3个点（线性可分）。

这里选取中心点为[1，0]，[0,1]，也会得到类似的结果。

使用工具箱函数

%% 清理
clear all;close all;clc

%% 输入
% 输入向量
P = [0,0,1,1;0,1,1,0]
%P =
%
%     0     0     1     1
%     0     1     1     0
% 期望输出
T = [0,1,0,1]
%T =
%
%     0     1     0     1
%% 建立网络
net=newrbe(P,T);

% 测试
out=sim(net,P)
%% 绘图
x=0:.2:1;
N=length(x);                             %  N=6

% X为新的输入向量
X(1,:)=reshape(repmat(x,N,1),N*N,1);
X(2,:)=repmat(x,1,N);
out2=sim(net,X);
% 整理为N*N矩阵
out2=reshape(out2,N,N);
% 绘图
mesh(x,x,out2);

执行后输出：

out =    -0.0000    1.0000     0    1.0000

用图形表示RBF网络对于[0,1]这一正方形区域的取值：

从图中可以看出，在（0,0）和（1,1）点处，函数的值为零，而在点（0,1）和（1,0）处，函数值为1。整体上取值从负开始变化到1。

实验2，曲线拟合

描述：用径向基网络拟合曲线

手算法

%% 清理
clear all;close all;clc

%%
% 输入
x=-9:8;                % 训练数据，间隔为1

% 期望输出;
y=[129,-32,-118,-138,-125,-97,-55,-23,-4,...
    2,1,-31,-72,-121,-142,-174,-155,-77];
% 隐含节点的中心
t=x;

% 计算每一个输入到每一个中心的距离，作为隐含层的输入
z=dist(x',t);
% 计算隐含层的输出 
G=radbas(z);
%期望输出
d=y';
% 伪逆。求出权值向量
w=inv(G.'*G)*G.'*d;

% 测试输入
xx=-9:.2:8;            % 测试数据，间隔为0.2

% 计算输入到中心的距离
t = x;
zz=dist(xx',t);

% 计算隐含层的输出
 GG=radbas(zz);

% 计算输出层的输出
Y=GG*w;

%% 绘图
% 原始数据点
plot(x,y,'o');
hold on;
% 拟合的函数曲线
plot(xx,Y','-');
legend('原始数据','拟合数据');
title('用径向基函数拟合曲线');

圆形坐标点为原始数据，直线为拟合的曲线。

将隐含层输入z、隐含层输出G的图画出来：

上面是z（2-范数，也就是据各个中心点的距离），下面是G（高斯函数输出）。可见，与xn距离越近，相应的radbas函数值就越大，距离xn远的，输出值为0。这一点体现了径向基神经网络的局部响应的原则，从G的图中可以看到，被激活的只有3个点，其余点都处于抑制状态。

不同于线性插值（拟合的过程就是一个插值的过程）的采用距离x最近的两个已知点乘以一定的权重来得到x处的函数值，在径向基网络中，取的点为所有的中心点，但是权重的下降由于径向基函数的原因是指数型的。下图说明了径向基插值的原理：

工具箱函数

%% 清理
clear all;close all;clc

%% 定义原始数据
x=-9:8;
y=[129,-32,-118,-138,-125,-97,-55,-23,-4,...
    2,1,-31,-72,-121,-142,-174,-155,-77];

%% 设计RBF网络
P=x;
T=y;
% 计时开始
tic;
% spread = 2
net = newrb(P, T, 0, 0.5);
% 记录消耗的时间
time_cost = toc;

%% 测试
% 测试数据
xx=-9:.2:8;

% 网络仿真
yy = sim(net, xx);

%%绘图
% 原数据点
figure;
plot(x,y,'o');
hold on;
% 仿真得到的拟合数据
plot(xx,yy,'-');
hold off;

% 图例、标题
legend('原始数据','拟合数据');
title('用径向基函数拟合曲线');

不同的spread值得到拟合曲线也不同，所以要选用合适的spread避免欠拟合和过拟合的出现。

spread = 0.2
spread = 0.5
spread = 2
spread = 5
