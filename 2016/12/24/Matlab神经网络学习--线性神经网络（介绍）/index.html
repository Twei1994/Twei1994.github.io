Matlab神经网络学习--线性神经网络（介绍）

介绍

线性神经网络与感知器的不同之处在于它采用了线性函数作为传输函数，所以输出可以取任意值。这个特性决定了线性网络可以用作实现拟合或逼近。线性神经网络采用Widrow-Hoff学习规则，也就是Δ rule来调整网络的权值和偏置，目标是使网络输出的LMS（Least Mean Square）最小均方误差最小。

如果网络包含多个输出节点，这种线性网络叫做Madaline网络。结构图如下：

它可以通过间接的方式解决线性不可分的问题，就是同多个线性函数对区域进行划分，然后对各个神经元的输出作逻辑运算。

线性神经网络解决线性不可分问题的另一个方法是，对神经元添加非线性输入，从而引入非线性成分，这样会使等效的输入维度变大，如下所示：

常用函数

newlind–设计一个线性网络

net = newlind(P, T, Pi)
  • P: R*Q的矩阵，包含Q个训练输入向量；
  • T: S*Q的矩阵，包含Q个训练输出向量；
  • Pi: 细胞数组，包含初始输入延迟；

函数返回设计好的线性网络net，也就是说已经得到了使结果均方误差最小的网络权重和偏置，不需要train来训练了。

newlin–构造一个线性层

net = newlin(P, S, ID, LR)
  • ID: 表示输入延迟的向量，默认值为[0]
  • LR: 学习率，默认值为0.01

在高版本的matlab中，用linearlayer代替newlin函数。

net = linearlayer(ID, LR)

learnwh–LMS学习函数

[dW, LS] = learnwh(W, P, Z, N, A, E, gW,gA, D, LP, LS)
  • W: 权值矩阵
  • P: 输入向量
  • Z: 权值输入向量
  • N: 网络输入向量
  • A: 输出向量
  • T: 期望输出向量
  • E: 层误差输出
  • gW: 权重梯度
  • gA: 输出梯度
  • D: 神经元距离
  • LP: 学习参数
  • LS: 学习状态，初始为空
  • dW: 权值变化矩阵
  • LS: 新的学习状态

重要的参数其实就只有输入P，误差E和学习率LP，通过type learnwh可以查看源函数代码。

仿真实验

实验1

描述： 用newlind创建一个线性网络，来拟合一个线性方程，线性回归

x=-5:5;
y=3*x-7;                    % 直线方程为 
randn('state',2);                % 设置种子，便于重复执行
y=y+randn(1,length(y))*1.5;            % 加入噪声的直线
plot(x,y,'o');
P=x;T=y;
net=newlind(P,T);                % 用newlind建立线性层
new_x=-5:.2:5;                    % 新的输入样本
new_y=sim(net,new_x);                % 仿真
hold on;plot(new_x,new_y);
legend('原始数据点','最小二乘拟合直线');
% 输出权重和阈值
net.iw                        % 权值为2.9219
% ans = 
%     [2.9219]
net.b                        % 偏置为-6.6797
% ans = 
%     [-6.6797]

title('newlind用于最小二乘拟合直线');

结果图如下：

可见，对增加了噪声的数据点的拟合结果为y = 2.9219x -6.6797，与原方程y = 3x -7大致相同。

实验2

描述： 内容与上一个实验相同，用newlin实现

x=-5:5;
y=3*x-7;                        % 直线方程为 
randn('state',2);        % 设置种子，便于重复执行
y=y+randn(1,length(y))*1.5;    % 加入噪声的直线
plot(x,y,'o');
P=x;T=y;
net=newlin(minmax(P),1,[0],maxlinlr(P));    % 用newlin创建线性网络
% net = linearlayer([0],maxlinlr(P));    % 用linearlayer创建线性网络
tic;
net=train(net,P,T);        % 训练。与newlind不同，newlin创建的网络需要调用训练函数
toc     % tic,toc用于计时
new_x=-5:.2:5;
new_y=sim(net,new_x);           % 仿真
hold on;plot(new_x,new_y);
legend('原始数据点','最小二乘拟合直线');
title('newlin用于最小二乘拟合直线');
% 训练好的网络的权重和偏置
net.iw
% ans =
%     [2.9219]    
net.b
% ans = 
%     [-6.6797]

其中，minmax函数用于将输入向量归一化，maxlinlr函数用于求最大学习率。

结果图如下：

拟合的结果与newlind是一致的。

实验3

描述：解决同一个问题，用learnwh实现一个线性神经网络。

%% 清理
clear,clc
close all

%% 定义数据
P=-5:5;                         % 输入：11个标量
d=3*P-7;
randn('state',2);    
d=d+randn(1,length(d))*1.5     % 期望输出：加了噪声的线性函数

P=[ones(1,length(P));P]        % P加上偏置
lp.lr = 0.01;                  % 指定学习率
% lp.lr = maxlinlr(P)            % 也可以用maxlinlr来计算
MAX = 150;                     % 最大迭代次数
ep1 = 0.1;                     % 均方差终止阈值
ep2 = 0.0001;                  % 权值变化终止阈值
%% 初始化
w=[0,0];

%% 循环更新
 for i=1:MAX
    fprintf('第%d次迭代：\n', i)
    e=d-purelin(w*P);          % 求得误差向量
    ms(i)=mse(e);              % 均方差
    ms(i)
    if (ms(i) < ep1)           % 如果均方差小于某个值，则算法收敛
        fprintf('均方差小于指定数而终止\n');
       break; 
    end

    dW = learnwh([],P,[],[],[],[],e,[],[],[],lp,[]);    % 权值调整量
    if (norm(dW) < ep2)        % 如果权值变化小于指定值，则算法收敛
       fprintf('权值变化小于指定数而终止\n');
       break;
    end
    w=w+dW                     % 用dW更新权值    

 end

%% 显示
fprintf('算法收敛于：\nw= (%f,%f),MSE: %f\n', w(1), w(2), ms(i));
figure;
subplot(2,1,1);                % 绘制散点和直线
plot(P(2,:),d,'o');title('散点与直线拟合结果');
xlabel('x');ylabel('y');
axis([-6,6,min(d)-1,max(d)+1]);
x1=-5:.2:5;
y1=w(1)+w(2)*x1;
hold on;plot(x1,y1);
subplot(2,1,2);                % 绘制均方差下降曲线
semilogy(1:i,ms,'-o');
xlabel('迭代次数');ylabel('MSE');title('均方差下降曲线');

结果输出为：

...
第78次迭代：ans = 2.7097

权值变化小于指定数而终止
算法收敛于：w= (-6.678867,2.921909),MSE: 2.709691

执行结果图如下：

可见，执行结果与之前的newlind和newlin是一样的。
设置了3个终止训练的条件，分别是迭代次数，均方误差和权值更新变化量。
