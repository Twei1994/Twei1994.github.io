Matlab神经网络学习--单层感知器（应用）

介绍

单层感知器是最简单的神经网络。通过对于坐标点的二分类问题，使用手算和工具箱两种方法验证单层感知器对于线性可分问题的应用性，并验证其对于线性不可分问题的局限性。

代码

手算代码（线性可分）

%% 清理
clear,clc
close all

%%
n=0.2;                  % 学习率
w=[0,0,0];                 % 初始权重
P=[ -9,  1, -12, -4,   0, 5; 15,  -8,   4,  5,  11, 9];  % 6个点
d=[0,1,0,0,0,1];        % 点的类别，期望输出

P=[ones(1,6);P];        % 增加一行全1，用于更新阈值
MAX=20;                 % 最大迭代次数为20次

%% 训练
i=0;
while 1
    v=w*P; 
    y=hardlim(v);       % 实际输出
    %更新
    e=(d-y);
    ee(i+1)=mae(e);
    if (ee(i+1)<0.001)   % 判断
        disp('we have got it:');
        disp(w);          % 输出训练后的权重
        break;
    end
    % 更新权值和偏置
    w=w+n*(d-y)*P';

    if (i>=MAX)         % 达到最大迭代次数，退出
        disp('MAX times loop');
        disp(w);
        disp(ee(i+1));
       break; 
    end
    i= i+1;
end


%% 显示
figure;
subplot(2,1,1);         % 显示待分类的点和分类结果
plot([-9 ,  -12  -4    0],[15, 4   5   11],'o');
hold on;
plot([1,5],[-8,9],'*');
axis([-13,6,-10,16]);
legend('第一类','第二类');
title('6个坐标点的二分类');
x=-13:.2:6;
y=x*(-w(2)/w(3))-w(1)/w(3);  % 绘制超平面曲线
plot(x,y);
hold off;

subplot(2,1,2);         % 显示mae值的变化
x=0:i;
plot(x,ee,'o-');
s=sprintf('mae的值(迭代次数:%d)', i+1);
title(s);

手算代码（线性不可分）

将点[-9, 15]的类别修改成为第二类，这样就变成了一个线性不可分的问题了。

%% 清理
clear,clc
close all

%%
n=0.1;                  % 学习率
w=[0,0,0]; 
P=[ -9,  1, -12, -4,   0, 5;...
   15,  -8,   4,  5,  11, 9];
d=[1,1,0,0,0,1];        % 期望输出

P=[ones(1,6);P];
MAX=100;                 % 最大迭代次数为100次
%% 训练
i=0;
while 1
    v=w*P; 
    y=hardlim(v);       % 实际输出
    %更新
    e=(d-y);
    ee(i+1)=mae(e);
    if (ee(i+1)<0.001)   % 判断
        disp('we have got it:');
        disp(w);
        break;
    end
    % 更新权值和偏置
    w=w+n*(d-y)*P';

    if (i>=MAX)         % 达到最大迭代次数，退出
        disp('MAX times loop');
        disp(w);
        disp(ee(i+1));
       break; 
    end
    i= i+1;
end


%% 显示
figure;
subplot(2,1,1);         % 显示待分类的点和分类结果
plot([  -12  -4    0],[ 4   5   11],'o');
hold on;
plot([-9 ,1,5],[15,-8,9],'*');
axis([-13,6,-10,16]);
legend('第一类','第二类');
title('6个坐标点的二分类');
x=-13:.2:6;
y=x*(-w(2)/w(3))-w(1)/w(3);
plot(x,y);
hold off;

subplot(2,1,2);         % 显示mae值的变化
x=0:i;
plot(x,ee,'o-');
s=sprintf('mae的值(迭代次数:%d)', i+1);
title(s);

实验结果

最终结果：MAX times loop:3.6000 1.0000 1.0000 0.3333

效果图如下：

结论：

  1. 从图中可见，这是一个线性不可分问题；
  2. 输出结果前三个值为权重和阈值，第四个是最后一次迭代误差；
  3. 感知器无法在有限的时间内收敛；
  4. 退出训练是由于迭代的次数大于了预设值100，但是误差处于振荡状态，不是稳定下降。

使用工具函数

% 清理
clear,clc
close all

% 创建感知器
net=newp([-20,20;-20,20],1);

%定义输入训练向量
P=[ -9,  1, -12, -4,   0, 5;...
   15,  -8,   4,  5,  11, 9];
% 期望输出
T=[0,1,0,0,0,1];

% 训练
net=train(net,P,T);

% 输入训练数据仿真验证
Y=sim(net,P);

% 绘制图案
w = [net.b{1},net.iw{1}]  % 获取权重和阈值
figure;
plot([-9 ,  -12  -4    0],[15, 4   5   11],'o');
hold on;
plot([1,5],[-8,9],'*');
axis([-13,6,-10,16]);
legend('第一类','第二类');
title('6个坐标点的二分类');
x=-13:.2:6;
y=x*(-w(2)/w(3))-w(1)/w(3);
plot(x,y);
hold off;

权重和阈值：[14, -6, 0]

效果图如下：

总结

线性感知器是最简单的神经网络，只适用于线性可分的问题中，功能有限。
